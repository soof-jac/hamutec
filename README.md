# HAMUTEC — Segurança Eletrónica

Site institucional profissional da **Hamutec**, empresa líder em segurança eletrónica sediada no Lubango, Angola.

## 🚀 Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 (Custom Properties), Vanilla JavaScript.
- **Gráficos 3D:** Three.js (Animação "Rede de Vigilância" no Hero).
- **Integrações:** API do WhatsApp para orçamentos e contactos.
- **Design:** Responsivo, Moderno (Dark Theme) e com animações de Scroll Reveal.

## 🛠️ Funcionalidades
- **Hero Interativo:** Fundo animado em 3D que reage ao movimento do rato.
- **Catálogo de Preços:** Sistema de abas para visualização rápida de pacotes (Alarmes, CCTV, Câmeras Wi-Fi, Cerca Elétrica).
- **Conversão Direta:** Botões de orçamento vinculados diretamente ao WhatsApp com mensagens pré-preenchidas.
- **Formulário de Contacto:** Validação e sanitização de dados antes do envio.
- **Equipa:** Galeria de profissionais com fotos reais.

## 📦 Como Abrir Localmente
1. Clone este repositório ou baixe os ficheiros.
2. Abra o ficheiro `index.html` em qualquer navegador moderno.
3. *Nota:* Para a animação Three.js carregar, é necessária uma ligação à internet (CDN).

## 🌐 Deployment (GitHub Pages)
Este site está pronto para ser hospedado no GitHub Pages:
1. Vá a **Settings** > **Pages** no seu repositório.
2. Selecione a branch `main` e a pasta `/(root)`.
3. Clique em **Save**.
